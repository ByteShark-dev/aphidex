# Creature Scanner backend

This folder contains the Firebase Functions backend for `scanCreature`.

## Required setup

1. Create or reuse the Firebase project for Aphidex.
2. Enable Cloud Functions in Firebase.
3. Enable **Cloud Vision API** in the linked Google Cloud project.
4. Replace `lib/firebase_options.dart` with the file generated by:

```bash
flutterfire configure
```

5. Authenticate Firebase CLI and deploy:

```bash
cd functions
npm install
npm run build
firebase deploy --only functions
```

## Current callable function

- `scanCreature`
  - Region: `us-central1`
  - Input: `{ imageBase64: string }`
  - Output:

```json
{
  "rawLabels": ["ladybug", "insect"],
  "rawWebEntities": ["ladybird", "ladybug"]
}
```

## Security notes

- No Vision API key is exposed in Flutter.
- The function is prepared for future App Check enforcement and rate limiting.
- Flutter already compresses images before upload and the backend also enforces a payload limit.
